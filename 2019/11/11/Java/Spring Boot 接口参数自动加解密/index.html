Spring Boot 接口参数自动加解密

2019-11-11 Java
  • Maven，pom.xml中添加依赖
  • 启动类Application中添加@EnableSecurity注解
  • 在application.yml或者application.properties中添加RSA公钥及私钥
  • 对返回值进行加密
  • 对传过来的加密参数解密
