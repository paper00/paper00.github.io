OSI 模型传输数据过程

数据发送


7层模型封包过程

  1. 应用层
    [源地址] [目的地址] [data]
  2. 表示层
    转化格式、加密
  3. 会话层
    建立 session、管理、维护
  4. 传输层
    + [TCP] 报文头 (字段 segment)
  5. 网络层
    + [IP] 报文头 (包 packet)
  6. 链路层
    + [以太] 报文头 (帧 frame)
    (fa:aa:14:91:3d:be) MAC 地址
  7. 物理层
    网卡 => 010101001 (转换成) 电信号
    (bits)

数据接收


7层模型解包过程

  1. 物理层
    网卡 => 电信号 (转换成) 010101001（二进制）
  2. 链路层
    交换机 => 以太信息
  3. 网络层
    + [IP] 报文头
  4. 传输层
    + [TCP] 报文头
  5. 会话层
    建立socket连接
  6. 表示层
    区分类型 TEXT JPEG PNG 等
  7. 应用层
    向 OS 要数据、并且显示
