python 实现

通过 python 的 keras 具体实现了预测算法。下面是具体步骤：

  1. 加载降雨量数据，对数据集进行划分；
  2. 数据归一化（提高模型训练的效率）；
  3. 编写Morlet小波函数；
  4. 建立LSTM模型，激励函数使用Morlet小波函数；
  5. 训练模型；
  6. 对模型进行评估，求MSE；
  7. 使用训练好的模型预测降雨量；
  8. 数据反归一化；
  9. 预测结果可视化；
